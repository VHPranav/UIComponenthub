## Overview

The Glass Button is designed to feel physical and layer-based, using standard Tailwind CSS backdrop filters. It works best on colorful or complex backgrounds where the transparency can shine.

## Features

- **Backdrop Blur**: 12px blur effect for realistic glass feel.
- **Hover Sweep**: A hidden gradient that sweeps across the button on hover.
- **Scale Interaction**: Subtle scaling on hover and active states.

## Installation

```bash
npm install lucide-react framer-motion
```
