# Animated Toggle Group

A set of two-state buttons that can be toggled on or off, featuring a premium sliding highlight effect for single-selection modes and distinct active states for multiple-selection modes.

## Features

- **Sliding Highlight**: In `single` selection mode, the active background smoothly slides between items using Framer Motion `layoutId`.
- **Radix UI Primitives**: Built on top of `@radix-ui/react-toggle-group` for robust accessibility and keyboard navigation.
- **Multiple Modes**: Supports both `single` and `multiple` selection types.
- **Context-aware Styling**: Variants and sizes can be set on the root `ToggleGroup` and are automatically inherited by all items.
- **Themed Container**: Includes a subtle background and border out-of-the-box for a "pill" look.

## Installation

```bash
npm install @radix-ui/react-toggle-group framer-motion lucide-react clsx tailwind-merge
```

## Props

### ToggleGroup Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `type` | `"single" \| "multiple"` | `required` | Determines if one or multiple items can be selected. |
| `variant` | `"default" \| "outline"` | `"default"` | The visual style of the group. |
| `size` | `"sm" \| "default" \| "lg"` | `"default"` | The size of the toggle items. |
| `value` | `string \| string[]` | `undefined` | The controlled value of the group. |

### ToggleGroupItem Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `value` | `string` | `required` | A unique value for the item. |
| `disabled` | `boolean` | `false` | Whether the item is disabled. |

## Usage

```tsx
import { ToggleGroup, ToggleGroupItem } from "@/components/ui/toggle-group"
import { AlignLeft, AlignCenter, AlignRight } from "lucide-react"

export default function ToggleGroupDemo() {
  return (
    <ToggleGroup type="single" defaultValue="center">
      <ToggleGroupItem value="left">
        <AlignLeft className="h-4 w-4" />
      </ToggleGroupItem>
      <ToggleGroupItem value="center">
        <AlignCenter className="h-4 w-4" />
      </ToggleGroupItem>
      <ToggleGroupItem value="right">
        <AlignRight className="h-4 w-4" />
      </ToggleGroupItem>
    </ToggleGroup>
  )
}
```
