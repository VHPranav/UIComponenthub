# Glare Card

The `GlareCard` component provides a high-fidelity interactive surface that reacts to mouse movement. It features a physics-based 3D tilt effect and a dynamic glare that simulates light reflecting off a polished surface.

## Preview

<ComponentPreview name="glare-card" />

## Installation

```bash
npm install framer-motion
```

## Usage

```tsx
import { GlareCard } from "@/registry/animations/glare-card"

export default function Example() {
  return (
    <GlareCard spotlight className="h-96 w-72 bg-zinc-950 p-8">
      <h3 className="text-xl font-bold">Interactive Surface</h3>
      <p className="text-zinc-500 mt-2">
        Hover over this card to see the 3D tilt and dynamic light reflection.
      </p>
    </GlareCard>
  )
}
```

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `children` | `ReactNode` | - | The content to be wrapped by the glare effect. |
| `className` | `string` | - | Additional CSS classes for the card container. |
| `glareColor` | `string` | `"rgba(255, 255, 255, 0.4)"` | The color of the glare effect. |
| `glareSize` | `number` | `150` | The size of the glare effect in percentage. |
| `glareOpacity` | `number` | `0.5` | The maximum opacity of the glare. |
| `tiltMax` | `number` | `20` | Maximum tilt rotation in degrees. |
| `perspective` | `number` | `1000` | Perspective distance in pixels. |
| `spotlight` | `boolean` | `false` | Enable radial spotlight mode instead of beam mode. |
| `glareAngle` | `number` | `45` | The angle of the glare gradient (beam mode only). |
