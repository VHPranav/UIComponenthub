# Magic Rings Serenity

<ComponentPreview id="magic-rings-serenity" />

The `MagicRingsSerenity` component renders a slow, deeply blurred and soft particle scene. It evokes a completely different atmosphere perfect for calm landing pages.

## Installation

This component depends on `three.js`. You will also need the core `MagicRings` component installed in your registry.

```bash
npm install three
npm install -D @types/three
```

## Usage

```tsx
import { MagicRingsSerenity } from "@/registry/backgrounds/magic-rings-serenity"

export default function Example() {
  return (
    <div className="relative w-full h-[500px]">
        <MagicRingsSerenity />
    </div>
  )
}
```
